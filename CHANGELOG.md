# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-03-21

### Added
- Initial release
- `read_memebers()` function to read and display member names from a CSV file
- CLI support with optional file path argument (defaults to `members.csv`)
- Error handling for missing file, permission errors, missing columns, and unexpected errors
- `pyproject.toml` for pip-installable packaging
- `read-members` CLI shorthand command after package installation
